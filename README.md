#PdTl abstractions

A timeline for Pure Data

--------------------------------

###requirements :
You'll need following Pd externals :

-	**moonlib**
-	**zexy**
-	**hcs**
-	**iemlib**
-	**ggee**

You'll also need [**AutoPreset**](https://github.com/MetaluNet/AutoPreset) abstractions.
